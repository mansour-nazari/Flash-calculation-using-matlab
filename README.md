# Multi-Method Flash Calculation Analysis (MATLAB)

A comparative study implementing and evaluating four distinct flash calculation algorithms for hydrocarbon systems, benchmarked against CMG WinProp results.


## Objective
To compare the numerical stability, convergence rate, and thermodynamic accuracy of different flash algorithms against the industry-standard CMG WinProp PVT simulator.


## Dependencies
- MATLAB (Optimization Toolbox required)
- CMG WinProp (for validation data)
